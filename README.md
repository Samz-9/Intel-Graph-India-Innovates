
# 🌍 Mini Global Intelligence Graph – India-Centric

![Next.js](https://img.shields.io/badge/Next.js-15-black)
![React](https://img.shields.io/badge/React-18-blue)
![Tailwind](https://img.shields.io/badge/TailwindCSS-UI-38B2AC) 

An **AI-powered geopolitical intelligence dashboard** that analyzes global signals using **real news + economic indicators** and visualizes relationships between major world powers.

The platform demonstrates how **AI + Knowledge Graphs + Data Visualization + Simulation** can help understand global geopolitical dynamics affecting **India and the world**.

---

# 🚀 Project Overview

**Mini Global Intelligence Graph** converts global signals into a visual intelligence system.

The platform processes:

* 🌐 Geopolitical news
* 📊 Economic indicators
* 🤝 Country relationships

It transforms this information into:

* 🧠 **Knowledge Graph**
* ⚠️ **Risk Intelligence Score**
* 🔮 **What-If Simulation Engine**

The result is a **clean and interactive dashboard** that helps visualize geopolitical risk.

# 🧠 Key Features

## 🌐 Knowledge Graph

A dynamic visualization of global relationships.

Nodes represent **countries** while edges represent:

* `trade_with`
* `alliance_with`
* `tension_with`

Example:

```
India → trade_with → USA
India → tension_with → China
Russia → energy_exports → India
```

---

## 📰 AI News Intelligence

Latest geopolitical headlines are analyzed using AI.

The AI extracts:

* Country involved
* Relationship context
* Sentiment

Example output:

```
India → tension_with → China
Sentiment: Negative
```

---

## ⚠️ Risk Score Engine

A simplified model estimates geopolitical risk.

```
Risk Score =
(Conflict News × 0.4) +
(Negative Sentiment × 0.3) +
(Inflation × 0.3)
```

| Score  | Risk Level |
| ------ | ---------- |
| 0–30   | 🟢 Low     |
| 31–60  | 🟡 Medium  |
| 61–100 | 🔴 High    |

---

## 🔮 What-If Simulation

Users can simulate economic scenarios and observe their impact.

Example:

**Oil price increase: +20%**

Impact:

* Inflation increases
* Risk score rises
* Economic pressure grows

This demonstrates **policy ripple effects across the system**.

---

# 🤖 Agent-Based Intelligence System

The platform uses a **multi-agent analysis system** called the **Cabinet of Five**.

| Agent          | Domain                | Insight Example                                   |
| -------------- | --------------------- | ------------------------------------------------- |
| Econ Agent     | GDP, Trade, Inflation | "Manufacturing may increase but inflation rises." |
| Security Agent | Geopolitics, Conflict | "Risk of trade retaliation."                      |
| Resource Agent | Energy, Climate       | "Energy costs will increase."                     |
| Tech Agent     | Infrastructure, AI    | "More digital infrastructure required."           |
| Society Agent  | Labor, Sentiment      | "Possible workforce resistance."                  |

An **Orchestrator Agent** combines insights from all agents into a final summary.
